# vue-chat

The chat application.
I made this to study Vue.js.

## Requirements

- node 12.10.0 or more
- npm 6.13.0 or more

## Setup

Clone this repository and install packages:

```
git clone https://github.com/swimpenguin/vue-chat.git
cd vue-chat
npm install
```

Then, you can run application.

```
npm run serve
```
